# Security Policy

Nocky is currently in public beta.

Please avoid publishing sensitive security details in a public issue. Contact the project maintainer through the GitHub profile associated with this repository and include a minimal reproduction when possible.

Do not include private music-library paths, account tokens or personal data in reports.

## YouTube Music sessions

Browser-session headers and cookies must be treated as credentials. Never attach them to public issues. Disconnect the account before sharing diagnostic archives, and redact `Authorization`, `Cookie`, `SAPISID` and Googlevideo URLs from logs.
